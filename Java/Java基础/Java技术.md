## Java 款平台原理

​	Java在不同操作系统上运行的原理是，在需要运行的Java程序上，安装一个与操作系统对应的Java虚拟机即可（JVM Java virtual machine)

JDK是Java程序开发工具包，包含JRE和开发人员使用的工具，其中包括：编译工具(javac.exe)和运行工具(java.exe)
