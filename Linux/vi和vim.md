# vi和vim

## vi和vim基本介绍

Linux系统会内置vi文本编辑器

Vim具有程序编辑的能力，可以看作是Vi的增强版本，可以主动的以字体颜色去辨识语法的真确性，方便程序设计，代码补全，编译和错误跳转的功能
