# 定时任务调度

## crond任务调度

* 概述

任务调度：是指系统在某个时间执行的特定的命令和程序

任务调度分类：

1. 系统工作
2. 有重要的工作必须周而复始地执行，如病毒扫描
3. 个别用户工作：个别用户可能希望执行某些程序，比如mysql数据库的备份

* 基本语法

crontab [选项]
